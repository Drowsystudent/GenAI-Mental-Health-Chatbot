# GenAI Mental Health Chatbot

Team: David Long, Avery Im, Brody Curry, Chase Wolverton, Clayton Tucker

## Quickstart (local)

### 1) Start the backend (Flask)

```bash
cd backend
python -m pip install -r requirements.txt
python app.py

```
### 2) start frontend using node.js
```bash
cd frontend
npm install
npm start
```
